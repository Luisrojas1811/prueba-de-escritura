# Prueba de Escritura Veloz

Test de velocidad de escritura hecho con Python y tkinter.

## ¿Qué hace?

Muestra una frase aleatoria y mide cuántos segundos tardás
en escribirla correctamente.

## Tecnologías

- `tkinter` — interfaz gráfica (incluido en Python)
- `time` — mide el tiempo de escritura (incluido en Python)
- `random` — selecciona una frase aleatoria (incluido en Python)

## Uso

```bash
python main.py
```

Se abre una ventana. Hacés clic en "Nueva frase", escribís
lo que aparece y cuando terminás te muestra cuánto tardaste.

## Nota

La frase debe escribirse exactamente igual, respetando
mayúsculas y espacios.
